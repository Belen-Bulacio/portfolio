Introducción

Git es un sistema de control de versiones distribuido.

Permite trabajar de forma colaborativa sin sobrescribir cambios.

Instalación

Podés descargar Git desde https://git-scm.com/.

Está disponible para Windows, Linux y MacOS.

Primeros pasos

Después de instalar Git, configurá tu identidad:

git config --global user.name "Tu Nombre"

git config --global user.email "tucorreo@example.com"

Comandos básicos

Algunos comandos esenciales son:

  • git init: crea un repositorio
  • git add: agrega archivos al staging
  • git commit: guarda los cambios
  • git status: muestra el estado
  • git log: historial de commits

Estos comandos son los más usados por principiantes.

Ramas y Merge

Git permite trabajar en distintas ramas.

Para crear una nueva rama usás:

git branch nueva-rama

Y para unirla con la rama principal:

git merge nueva-rama

Usá git checkout para cambiar entre ramas.
